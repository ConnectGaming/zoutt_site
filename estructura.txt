ARCHIVOS A REVISAR / LIMPIAR:
=============================
⚠ assets/img/og-image.jpg           → DUPLICADO (usar /assets/meta/og-image.jpg)
⚠ assets/img/x-icon                 → ARCHIVO SIN EXTENSIÓN (eliminar)

CONFIGURACIÓN SEO Y REDIRECCIONES:
==================================
✓ .htaccess                         → URLs limpias (elimina .html de URLs)
✓ _redirects                        → Configuración Netlify para redirecciones
✓ robots.txt                        → Directivas crawling para motores búsqueda
✓ sitemap.xml                       → Mapa de URLs indexable
✓ manifest.json                     → PWA manifest con icono favicon
✓ seo/schema.json                   → Datos estructurados JSON-LD

META TAGS Y SOCIAL MEDIA:
=========================
✓ Open Graph (og:image)             → 1200x630px en /assets/meta/og-image.jpg
✓ Twitter Card                      → Mismo og-image.jpg
✓ Favicon                           → /favicon.ico (32x48px)

REFERENCIAS EN ARCHIVOS:
=======================
index.html:
  - og:image → /assets/meta/og-image.jpg ✓
  - twitter:image → /assets/meta/og-image.jpg ✓
  - favicon → /favicon.ico ✓

manifest.json:
  - icons[0].src → /favicon.ico
